DBD Twitch Chat Overlay

1. Открой index.html.
2. Замени:
   const channel = "YOUR_TWITCH_CHANNEL";
   на свой логин Twitch.

Например:
const channel = "larriion";

3. Загрузите папку на GitHub Pages или Netlify.
4. Полученную ссылку вставьте в OBS:
   Источник -> Browser Source.

Размер рекомендуется:
Ширина 550
Высота 900
Фон прозрачный.
